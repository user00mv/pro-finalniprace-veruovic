# Osobní Portfolio - Marko Veruovič

Tento projekt je semestrální práce vytvořená jako moderní webové portfolio.

## Technologie
* HTML5, CSS3 (Bento Grid, Flexbox)
* JavaScript (Smooth scroll, Carousel, Drag-to-scroll)
* FontAwesome (Ikony)

## Sledování návštěvnosti
Vzhledem k tomu, že **GitHub Pages** je statický hosting a nepodporuje **PHP**, je sledování návštěvnosti (původně zamýšlené do logs/visits.csv) implementováno pomocí **HITS API**. Aktuální počet zobrazení lze vidět v patičce každé stránky.
